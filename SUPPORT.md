# Support

Use GitHub issues for bug reports, feature requests, and publishing questions:

https://github.com/marcmalerei/agent-flow/issues

When reporting a problem, include:

- VS Code version
- Agent Flow version
- Operating system
- Steps to reproduce
- Relevant `.github` file snippets with secrets removed

Agent Flow manages local workspace files only. Do not include private repository content, credentials, tokens, or proprietary prompts in public issues.
